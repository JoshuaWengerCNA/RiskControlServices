Application to automatically compare values of SOV and JOL data sets an provide matches where found.
Takes .xlsx SOV and JOL Location Report as input and generates a .xlsx with reconciliation results.
Implementes a simple GUI for easy use by less technical staff members.
Author: Joshua Wenger - EB Risk Control Specialist - joshua.wenger@cna.com
